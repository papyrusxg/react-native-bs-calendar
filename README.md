# react-native-bs-calendar

A Nepali (Bikram Sambat) date picker and calendar for a React Native / Expo app

## Installation


```sh
npm install react-native-bs-calendar
```


## Usage


```js
import { multiply } from 'react-native-bs-calendar';

// ...

const result = multiply(3, 7);
```
